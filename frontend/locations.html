Pinging Locations

Going to www.pinging.net will route you to our nearest available server.

You can also force using a server in a particular location. We currently have servers in:

  • Council Bluffs, Iowa, USA (Google Cloud)
  • Paris, France (Vultr Cloud)
  • Singapore (Vultr Cloud)

Individual locations do not have redundancy so for best reliability use www.pinging.net.

If having a server closer to you would be helpful please get in touch.
