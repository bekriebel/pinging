About Pinging

An unreliable internet connection...
    is so...
      frustrating!

Pinging.net quickly determines if you are online by running multiple tests. It then continues to monitor your connection via repeated tests including a "web ping" every second to help identify intermittent network issues.

Web Ping

Every second a WebRTC message is sent and returned by the server. UDP is used for the underlying transport mechanism so dropped messages are not retried which is desired in order to determine packet loss. Dropped packets will show up as red squares. The round trip time of each ping is graphed and can be viewed in the log or by hovering over the graph.

HTTP Test

Makes a HTTP POST request with a random string body and verifies it is returned properly. Caching of the request is also explicitly disabled.

HTTP is the foundational transportation technology used on the web.

DNS Test

A HTTP GET request to a [random-number]dns-check.pinging.net. Caching is explicitly disabled and the random number part of the subdomain prevents common caching of the DNS lookup.

DNS is the system that converts domain names (e.g. pinging.net) to an address that a computer understands (e.g. 192.168.100.100).

Browser Check

Web browsers report if they believe you are offline. If this check is failing it means you are likely not connected to your WiFi, cellular connection or ethernet cord. This is a basic check and it will often pass even if you are not connected. This check is continuously monitored by your browser.

Initial Load Check

Caching is disabled for this site so if this site loads it is a strong indication your internet is working at the time of loading the site.

More Information

Speed is also an important attribute of your internet connection. It is not currently monitored by this site but can be measured on sites such as fast.com or speedtest.net.

We have servers in several locations around the world.

Privacy

We do not collect personal data and do not use cookies. IP addresses are only stored for up to 72 hours. Metrics for each server host are publicly available.
